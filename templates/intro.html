Introduction

This application uses machine learning and computer vision to detect natural disasters in real-time.

The system leverages a pre-trained model to classify images captured via webcam as Cyclone, Earthquake, Flood, or Wildfire.

Real-time data from your webcam is processed, and the system attempts to predict the disaster type based on the visual input.

Click "Start Detection" to begin using the system.
